question:The product of a number N and eight less than N is -7. What is the sum of all possible values of N?

In this case, N = 1.Case 2: N-7 = 0. In this case, N = 7.So, the possible values of N are 1 and 7.We need to find the sum of all possible values of N.Sum of possible values of N is 1 + 7 = 8.We can check our answers by plugging them back into the original equation N(N-8) = -7.If N=1, then N-8 = 1-8 = -7. So N(N-8) = 1 times (-7) = -7. This is correct.If N=7, then N-8 = 7-8 = -1. So N(N-8) = 7 times (-1) = -7. This is correct.Thus, the possible values of N are 1 and 7.The sum of all possible values of N is 1+7=8.Alternatively, we can use the quadratic formula to solve N^2 - 8N + 7 = 0.The quadratic formula for ax^2 + bx + c = 0 is x = frac{-b pm sqrt{b^2 - 4ac}}{2a}.In our case, a=1, b=-8, c=7, and the variable is N.So, N = frac{-(-8) pm sqrt{(-8)^2 - 4(1)(7)}}{2(1)} = frac{8 pm sqrt{64 - 28}}{2} = frac{8 pm sqrt{36}}{2} = frac{8 pm 6}{2}.We have two possible values for N:N_1 = frac{8 + 6}{2} = frac{14}{2} = 7.N_2 = frac{8 - 6}{2} = frac{2}{2} = 1.So, the possible values of N are 1 and 7.The sum of all possible values of N is 1 + 7 = 8.For a quadratic equation ax^2 + bx + c = 0, the sum of the roots is given by -b/a.In our equation N^2 - 8N + 7 = 0, we have a=1, b=-8, c=7.The sum of the roots (possible values of N) is -b/a = -(-8)/1 = 8/1 = 8.The product of the roots is c/a = 7/1 = 7.The sum of possible values of N is 8.The possible values of N are 1 and 7.
