This LIFO behavior immediately suggests a **Stack**.4. **Algorithm Design (Stack-based approach):** * **Iterate through the String:** Process the input string character by character. * **Handling Opening Characters:** If the character is an opening parenthesis ( `(`, `{`, or `[` ), push it onto the stack.
